# Pdf2Img
This program exporting images from pdf file.

![pici](https://user-images.githubusercontent.com/51339282/151941800-f975e28c-8e7b-4e78-92a7-d33f30b51981.PNG)



## Supported Output formats:
- Bmp 
- Jpeg
- Gif 
- Tiff
- Png

## DPI
- Low - 72 
- Medium - 300
- High - 1000

# Language
- C# .net freamwork 4.5
