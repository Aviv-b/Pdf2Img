# Pdf2Img
This program exporting images from pdf file.

![pici](https://user-images.githubusercontent.com/51339282/151941984-a283a5d5-22fd-4515-bb85-a685c52c7d2e.PNG)



## Supported image formats:
- Bmp 
- Jpeg
- Gif 
- Tiff
- Png

## DPI
- Low - 72 
- Medium - 300
- High - 1000

# Language
- C# .net freamwork 4.5
